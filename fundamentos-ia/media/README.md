# Media Slots

Esta carpeta queda preparada para que despues subas el material de la clase.

Sugerencia de estructura:

- `slides/`
- `videos/`
- `papers/`
- `links/`

Ideas de uso:

- En `index.html`, reemplaza los bloques `.slide-media.placeholder` por imagenes reales o inserta un `<img>` dentro.
- En las secciones de video, completa `data-video-src` con una ruta como `media/videos/video-01.mp4`.
- En `recursos.html`, podes ir cambiando los items placeholder por links reales, PDFs o embeds.
