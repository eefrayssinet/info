# info

Experiencia web con dos rutas principales:

- `/` : landing de `Prompt Golf`
- `/fundamentos-ia` : landing de la clase `Fundamentos Basicos de IA`
- `/fundamentos-ia/recursos` : biblioteca de recursos

## Stack

- React
- Vite
- React Router

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
